Chapter 7 Pattern for matching \\:

Now if we want a pattern to detect the characters \\, we need to express each \. Remember to express one, we need \\. Since we want to express another, we need another \\. Hence to express two literal \’s, we need \\\\.

In sum, the pattern \\\\ matches the characters \\. Since \\ represents a single, literal \, \\\\ matches a single literal \.

Learning by example, the first str_detect below will not work because the pattern in str_detect(pattern) is not representing two slashes; it is representing one.

name <- "Adam \\Sadowski"

name %>% str_detect("Adam \\Sadowski")
>> [1] FALSE
name %>% str_detect("Adam \\\\Sadowski")
>> [1] TRUE
## Matching multiple slashes

# In line 28 there is only one literal slash: an escaped slash
# Hence
name %>% str_detect("\\\\{2}")
>> [1] FALSE
# is false

One slash is reserved for Special Metacharacters like \n, \t. Two slashes is reserved for Character Classes and Achors
